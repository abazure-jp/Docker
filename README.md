### How to use jwilder/nginx-proxy

1. `docker run -d -p 80:80 -v /var/run/docker.sock:/tmp/docker.sock:ro jwilder/nginx-proxy`
2. `docker run -e VIRTUAL_HOST my.app myapp`
3. Add `127.0.0.0.1 localhost my.app` in `/etc/hots`
